# Hướng Dẫn Sử Dụng NTTT-Proxy

## Giới Thiệu
NTTT-Proxy là một công cụ hỗ trợ lấy proxy miễn phí và quản lý proxy cho các mục đích kiểm thử, nghiên cứu hoặc phát triển phần mềm.

## Cách Sử Dụng
1. **Cài đặt**:
   - Chạy file `Setup.sh` (Linux/macOS) hoặc `SetupLinux.sh` để cài đặt các gói cần thiết.
   - Đảm bảo bạn có quyền thực thi cho các file `.sh` bằng lệnh: `chmod +x <tên_file.sh>`

2. **Khởi động**:
   - Chạy script `FreeProxy.sh` hoặc `src/Free-Proxy.sh` để lấy proxy miễn phí.
   - Có thể sử dụng `menu.py` để thao tác qua giao diện dòng lệnh.

3. **Quản lý proxy**:
   - Các proxy thu thập được sẽ lưu trong file `proxy.txt` hoặc `packages/Proxy.txt`.
   - Sử dụng các script trong thư mục `src/` để thực hiện các thao tác nâng cao.

## Miễn Trừ Trách Nhiệm
- NTTT-Proxy chỉ phục vụ mục đích học tập, nghiên cứu và kiểm thử.
- Người dùng tự chịu trách nhiệm với mọi hành động sử dụng proxy từ công cụ này.
- Không sử dụng NTTT-Proxy cho các mục đích vi phạm pháp luật hoặc gây hại cho hệ thống khác.
- Tác giả không chịu trách nhiệm với bất kỳ thiệt hại nào phát sinh từ việc sử dụng công cụ.
